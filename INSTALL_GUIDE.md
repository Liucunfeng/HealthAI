# 把「AI 健康助手」安装到安卓手机

> 重要前提：本工程是**源码**，需要先编译成 APK 才能装到手机。本机（WorkBuddy 运行环境）没有 Android SDK / 模拟器，**无法在此直接编译**，请用你自己的电脑完成下面步骤。

---

## 一、前置准备

| 项 | 说明 |
|---|---|
| 一台电脑 | Windows / macOS / Linux 均可 |
| Android Studio | 官方 IDE，下载：https://developer.android.com/studio |
| 一根数据线 | 能连手机和电脑（最好原装线） |
| 一部安卓手机 | Android 7.0（API 24）及以上 |
| 联网 | 首次打开工程会下载 Gradle 与依赖库 |

---

## 二、手机开启「USB 调试」

1. 打开手机 **设置 → 关于手机**
2. 连续点击 **「版本号」7 次**，提示"已进入开发者模式"
3. 返回设置，进入 **系统 / 更多设置 → 开发者选项**
4. 打开 **「USB 调试」**（弹出提示点"确定"）

> 部分国产手机还需打开「USB 安装」「"仅充电"模式下允许 ADB 调试」等开关。

---

## 三、方法一：Android Studio 直连安装（最简单）

1. 电脑打开 **Android Studio → Open → 选择 `HealthAIApp/` 目录**
2. 首次会自动下载 Gradle 与依赖，等状态栏进度走完（需联网）
3. 手机用数据线连电脑，手机上点 **「允许 USB 调试」**，并在弹窗选 **「传输文件(MTP)」**
4. Android Studio 顶部设备下拉应识别到你的手机；点绿色 ▶ **Run** 按钮
5. 编译完成后 APP 自动安装并启动

---

## 四、方法二：生成 APK 手动安装（可离线发给别人）

1. Android Studio 打开工程，等依赖就绪
2. 菜单 **Build → Build Bundle(s) / APK(s) → Build APK(s)**
3. 编译完右下角提示，点 **locate** 找到文件：
   `app/build/outputs/apk/debug/app-debug.apk`
4. 把这个 APK 通过微信/数据线/网盘传到手机
5. 手机打开 APK，按提示**允许「未知来源」安装**即可

> 这是 **debug 包**，无需签名，方便调试；正式发布请用 **Build → Generate Signed Bundle / APK** 生成带签名的 release 包。

---

## 五、常见坑

- **设备下拉里看不到手机**：重插数据线、换 USB 口、确认 USB 调试已开、装好手机驱动（Windows 常见）。
- **第一次打开工程很慢 / 报错**：多为 Gradle 或依赖下载失败，检查网络，必要时重试或挂代理。
- **安装被拦截**：在设置里允许「未知来源应用」对该文件管理器的安装权限。
- **API Key 未填**：进 APP「设置」填视觉 API Key 才能用拍照分析，否则会报网络/鉴权错误。

---

## 六、没有本地电脑编译？备选方案

- **借用同事/朋友的电脑**装 Android Studio 跑一次（最快）。
- **GitHub Actions 云编译**：把工程推到 GitHub，用 CI 自动产出 APK，下载即可装（无需本地 SDK）。
- **付费云构建服务**：如 Codemagic 等。

装好、填好 Key、建好档案后，即可拍照做身材 / 食物分析。
